Começando

Configure o CDW-FW em poucos passos e escolha entre UMD ou ESM conforme o seu fluxo.

Índice desta página UMD ESM

UMD (script)

Carregue os arquivos prontos da pasta dist e inicialize o namespace global. Quando usar: projetos sem bundler ou protótipos rápidos.

dist/cdw-fw.css dist/cdw-fw.js CDWFW.init()
Incluindo via script tag
Básico
Carregue CSS + JS e execute CDWFW.init().
O namespace global fica disponível como CDWFW.
HTML
<link rel="stylesheet" href="/dist/cdw-fw.css" />
<script src="/dist/cdw-fw.js"></script>
<script>
  CDWFW.init();
</script>

ESM (module)

Use o bundle ESM para importar apenas o necessário. Quando usar: projetos modernos com bundler.

dist/cdw-fw.esm.js import { init }
Importando como módulo
Comum
Importe init() e execute assim que a página carregar.
Ideal para integrar com Vite ou Webpack.
HTML
<script type="module">
  import { init } from "/dist/cdw-fw.esm.js";
  init();
</script>
